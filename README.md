
# Project 10: Text to Knowledge by GPT2

This repository contains all relevant codes except train.bin/val.bin about t2kg project.

"simple-wikidata-db" is used to extract raw labels, relational data from Wikidata dumps.

Other relevant training data preparing codes like sentence, triple, hops and paragraph generation are contained in "codes used through the whole process".

GPT2 model and relevant training/sampling codes are contained in "nanoGPT".
